# Apple Pay JS Check

Static React site for checking whether the current browser environment appears ready for Apple Pay.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub Pages deployment

This repo is configured for GitHub Pages project-site deployment at:

`https://shn2016.github.io/apple-pay-js-check/`

Deployment is handled by the GitHub Actions workflow in `.github/workflows/deploy.yml`.

To enable it in GitHub:

1. Open the repository settings.
2. Go to `Pages`.
3. Set the source to `GitHub Actions`.
4. Push to `main`.
